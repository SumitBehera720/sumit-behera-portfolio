This **Developer Portfolio Website** was created to showcase my **projects, technical skills, and development journey**.

The goal of this portfolio is to present my work in a **clean, modern, and professional way** so recruiters, developers, and collaborators can explore the projects I have built and the technologies I work with.

This project focuses on **simplicity, responsiveness, and strong visual design** while keeping the code clean and well-structured.
